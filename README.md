# Rock Paper Scissors Game (Python)

## 📌 Project Description

This is a simple Rock Paper Scissors game developed using Python where the player competes against the computer. The computer randomly selects rock, paper, or scissors and the program determines the winner based on standard game rules. This project demonstrates basic Python programming concepts.

## 🎯 Objective

* To create a simple Python game.
* To understand conditional statements.
* To practice loops and user input.
* To use the random module in Python.

## 🛠 Tools Used

* Programming Language: Python
* Software: Python IDLE / VS Code
* Operating System: Windows

## ⚙️ Features

* Player vs Computer gameplay
* Random computer choice
* Winner detection
* Simple user input system
* Repeat game option

## 📂 Project Structure

```
rock-paper-scissors-python

```

## ▶️ How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the Python file:

```
python game.py
```
